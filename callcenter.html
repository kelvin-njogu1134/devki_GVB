Dashboard Instructions

This dashboard displays all emergency alerts sent from the public LindaJamii emergency page.

How to use:

  • Each alert card shows the date, time, and simulated location in Ruiru area
  • Click the "View on Google Maps" link to see the simulated location on a map
  • Use the Refresh button to update the alert list
  • Use the Simulate Alert button to generate test alerts for demonstration
  • Use the Clear button to remove all alerts (for demonstration purposes)

Note: This is a demonstration system developed by Devki High School for KSEF. All locations are simulated around Ruiru area.
